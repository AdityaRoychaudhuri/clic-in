# clic-in

A full-stack telemedicine platform built with Next.js that brings the clinic experience online. It enables secure authentication, appointment scheduling, and real-time doctor–patient video consultations.

## 🚀 Features

- 🔐 User Authentication (Clerk)
- 👨‍⚕️ Role-based access (Doctor / Patient)
- 📅 Appointment booking system
- 🎥 Real-time video consultations
- 🧭 Dynamic video rooms
- 📊 Dashboard for managing sessions
- 🔒 Protected routes and secure access

## 🛠️ Tech Stack

- **Frontend:** Next.js, React, Tailwind CSS
- **Backend:** Next.js API Routes
- **Authentication:** Clerk
- **Video SDK:** Vonage 
- **Database:** Prisma
